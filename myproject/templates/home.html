Welcome! This is a simple Django application with REST API deployed on AWS Elastic Beanstalk.

Available API Endpoints:

GET /health/
Health check endpoint
GET /books/
List all books
POST /books/
Create a new book
GET /books/{id}/
Get a specific book
PUT/PATCH /books/{id}/
Update a specific book
DELETE /books/{id}/
Delete a specific book
GET /authors/
List all authors (supports similar CRUD operations)
GET /stats/
Get book statistics

Test the API:

You can test the API using tools like:

  • curl commands
  • Postman
  • Browser (for GET requests)
  • Django REST Framework browsable API

Example: Try visiting /health/ to see the health check response!
